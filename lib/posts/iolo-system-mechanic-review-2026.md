System Mechanic is a Windows PC optimization suite by Iolo Technologies (founded 1998, Los Angeles). Core areas:

## Core Features

### 1. PC Accelerate (One-Click Scan)

Covers junk files, registry, startup programs, memory, and internet settings in 2–3 minutes. One-click fix. Finds what Windows' built-in Disk Cleanup misses: Chrome/Edge cache, old Windows Update files, crash logs, thumbnail caches.

### 2. ActiveCare (Background Maintenance)

Runs maintenance tasks when your PC is idle. Pauses immediately when you start working. This is what separates System Mechanic from one-time cleanup tools — continuous maintenance vs manual cleanup every few months.

### 3. Startup Optimizer

Shows every startup program with a plain-English description and impact rating (high/medium/low). Far more useful than Windows Task Manager which just shows cryptic process names.

### 4. LiveBoost

Three system tray tools for on-demand performance: RAMJet (flushes cached RAM), CPU Boost (lowers background process priority), DriveAccelerator (optimizes disk operations).

### 5. Privacy Cleaner

Clears browsing history, form data, and downloads across Chrome, Firefox, Edge, and IE.
