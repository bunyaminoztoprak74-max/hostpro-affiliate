## Implementation Method

Each donor post received a `## Related Reading: [Topic]` section appended to its markdown content. Links use keyword-relevant anchor text (not generic 'click here' or title text) to maximize topical signal to Google.

### Anchor text principles used:
- **Keyword-first:** anchors lead with the target keyword (e.g. *'fastest web hosting by budget tier 2026'*)
- **Descriptive suffix:** adds context differentiator (e.g. *'Cloudways vs WP Engine: how Kinsta compares to both'*)
- **No exact-match over-optimisation:** variation in phrasing across multiple donors to same target
- **No generic anchors:** zero instances of 'click here', 'read more', or 'this article'
